# Narein Karthik E — Trainer Portfolio

A responsive single-page portfolio for Narein Karthik E, AI & Data Analytics Trainer and Founder of DTEKSKILZ.

## Local setup

1. Install Node.js 22 or newer.
2. Run `npm install`.
3. Run `npm run dev` and open the local URL shown in the terminal.

## Content and images

- Edit portfolio copy, lists and programme information in `app/data.ts`.
- Replace project placeholders in `app/portfolio.tsx` when approved screenshots are available.
- Portfolio photographs and branding are stored in `public/images`.
- Replace the contact placeholders only when verified phone and profile links are available.

## Production

Run `npm run lint` and `npm run build`. The project includes Sites hosting configuration in `.openai/hosting.json` and can be deployed through OpenAI Sites.
